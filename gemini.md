# GreenCart 購物清單系統開發全紀錄

## 專案概述
GreenCart 是一款專為現代使用者打造的極簡主義購物清單 Web 應用。本專案以使用者體驗為核心，結合了優雅的視覺設計與強大的實用功能，並以指定的品牌色 **GreenCart Green (#3AA94A)** 作為整體設計靈魂。

## 技術規格
- **前端架構**: React 18 (透過 UMD 模式 CDN 載入，免安裝環境)
- **腳本解析**: Babel Standalone (支援即時 JSX 轉譯)
- **資料持久化**: 瀏覽器原生 LocalStorage API (資料 100% 儲存在使用者本地端)
- **部署模式**: Single-File Architecture (所有邏輯、樣式、資源皆封裝於單一 `index.html`)
- **字體系統**: 整合 Google Fonts (Inter) 與多語系備援字型 (Microsoft JhengHei, Microsoft YaHei, PingFang)

## 核心功能詳解

### 1. 智慧型購物清單管理
- **分類系統**: 提供「生鮮食品、家居百貨、個人護理、電子產品、其他」五大預設分類。
- **動態排序算法**: 
    - 採用兩段式排序：未完成項目依時間戳記遞減排序（最新在前），已完成項目自動沉底。
    - 勾選完成時觸發平滑的視覺過渡與重新排序。
- **快速操作**: 支援 Enter 鍵快速新增與單鍵清除所有已完成項目。

### 2. 進度追蹤與回饋
- **即時進度條**: 頂部視覺化進度條隨勾選狀態即時滑動。
- **數據統計**: 清楚顯示「已完成項目數 / 總數」及百分比。

### 3. 多語言國際化支援 (I18N)
系統具備深度本地化能力，支援五種語言：
- **繁體中文**: 針對台灣/港澳習慣優化用語。
- **簡體中文 (zh-CN)**: 
    - **動態語言標記**: 切換時自動更改 `<html lang="zh-CN">`。
    - **字型渲染優化**: 強制瀏覽器調用簡體專用字型，避免筆畫顯示錯誤（如「清」、「單」等字）。
    - **用語本地化**: 例如將「新增」改為「添加」。
- **English**: 標準國際英語介面。
- **日本語**: 符合日本使用者習慣的分類與用語。
- **Deutsch**: 完整支援德文長單字佈局。
*註：切換語言時，網頁標題 (Title) 與檔案匯出名稱會同步更新。*

### 4. 主題美學系統 (Theming)
- **淺色模式 (Light Mode)**: 以 `#F4F7F5` 為背景，營造清新、乾淨的購物氛圍。
- **深色模式 (Dark Mode)**: 
    - 採用 `#121212` 高對比低反光背景。
    - 精準調整陰影與卡片色差，確保在低光源下依然維持絕佳的層次感。
- **狀態記憶**: 主題選擇會自動儲存於本地，使用者下次開啟時無需重新設定。

### 5. 數據匯出與實體列印
- **💾 檔案匯出**: 支援匯出為標準 `.txt` 格式，內容包含自動產生的時間戳記與勾選狀態。
- **🖨️ 專業列印模式**: 內建專用 `@media print` CSS。
    - 自動隱藏輸入框、切換按鈕及刪除按鈕等「非內容」元素。
    - 優化頁面佈局，確保清單在 A4 紙張上清晰易讀。

## 視覺與介面規範
- **主色調 (Primary)**: `#3AA94A`
- **深色強調 (Primary Dark)**: `#2E863B`
- **淺色背景 (Primary Light)**: `#EBF6ED`
- **邊框圓角 (Border Radius)**: `12px`
- **過渡動畫 (Transition)**: `all 0.3s ease`
- **響應式斷點**: `600px` (手機版自動轉為垂直佈局)

## 使用與部署建議
本系統不需要任何開發環境（Node.js 等）。使用者僅需確保電腦連接網路（以便初次載入 React 引擎），直接點擊 `index.html` 即可立即開始管理購物清單。
